# Day 26 – Prior Authorization Workflow Simulator

## Objective
Built an interactive simulator for the healthcare prior authorization workflow.

## Features
- Patient scenarios
- Drag-and-drop workflow
- Document assembly
- Approval/Pend/Denial/Appeal outcomes
- Journey tracker
- Educational explanations

## Learnings
- Understood the prior authorization process.
- Learned how insurance approvals work.
- Practiced building an interactive HTML application.
