# Day 39 - PDF Splitter & Merger

## 📌 Project Overview

Today, I developed a **PDF Splitter & Merger**, a fully client-side web application that allows users to split and merge PDF documents directly in the browser without uploading files to any server. The application provides a modern, responsive interface with smooth interactions and ensures complete user privacy by performing all processing locally.

---

## 🚀 Features Implemented

### PDF Splitter

* Upload PDF files
* Automatically detect total pages
* Generate page thumbnails
* Preview uploaded PDF
* Split by page numbers
* Split by custom page ranges
* Split after specific pages
* Split every N pages
* Extract selected pages
* Create multiple split ranges
* Validate page ranges
* Preview output structure before processing

### PDF Merger

* Upload multiple PDF files
* Drag-and-drop file upload
* Drag-and-drop file reordering
* Display page count for each file
* Preview uploaded PDFs
* Show total files and total pages
* Merge PDFs into a single document
* Download merged PDF

### Additional Features

* Dark Mode
* Responsive Design
* Glassmorphism UI
* Loading animations
* Processing indicators
* Keyboard shortcuts
* Accessibility support
* Error handling and validation
* Offline functionality after initial load
* Local browser-based processing (No backend)
