# Day 2 - Prompt Engineering

## What I Did
- Created an image using a simple prompt.
- Created another image using a detailed prompt.
- Compared both outputs and noted the differences.

## Key Learnings
- Better prompts give better results.
- Detailed instructions improve clarity and structure.
- Prompt engineering helps AI generate more useful outputs.

## Prompts Used
**Lazy Prompt:**  
Create an image explaining Prompt Engineering.

**Engineered Prompt:**  
Create a professional infographic explaining Prompt Engineering with examples, benefits, and best practices.

## Reflection
Today I learned that even small changes in a prompt can significantly improve the quality of AI-generated content.
