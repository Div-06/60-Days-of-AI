## Personalized Strategy

### Recommended Primary Model

**Claude Sonnet**

### Why This Model Fits Me

Claude Sonnet provides the best balance between speed, accuracy, coding assistance, learning support, and project planning. It is ideal for students who regularly work on coding projects, AI learning, research, and career preparation.

### When to Use Haiku

* Quick questions
* Fast summaries
* Brainstorming ideas
* Simple explanations
* Daily productivity tasks

### When to Use Sonnet

* Coding assistance
* DSA problem solving
* AI/ML learning
* Project development
* Resume improvement
* LinkedIn content creation
* Research and documentation

### When to Use Opus

* Deep research projects
* Complex system design
* Startup or business planning
* Advanced technical analysis
* Long-form strategic planning

### Recommended Effort Level

#### Low

Use for:

* Quick answers
* Definitions
* Simple summaries

#### Standard

Use for:

* Daily coding help
* Learning concepts
* Project planning
* Resume reviews
* Content creation

#### High

Use for:

* Debugging difficult issues
* Complex coding tasks
* Research analysis
* Architecture discussions

#### Max

Use for:

* Deep technical research
* Large project planning
* Advanced AI/ML concepts
* Comprehensive reports and strategies

## Biggest Mistakes I Should Avoid

* Using advanced models for simple tasks unnecessarily.
* Asking vague questions without sufficient context.
* Not specifying the desired output format.
* Relying on AI without verifying important information.
* Using Max effort for routine tasks that only require Standard effort.

## Final Recommendation

If I could use only **ONE model** and **ONE effort level** for most of my work:

**Model:** Claude Sonnet
**Effort Level:** Standard

### Why?

Claude Sonnet with Standard effort provides the best balance of speed, quality, reasoning, coding support, learning assistance, and productivity. It is the most practical choice for a student focused on coding, AI learning, project development, research, and career preparation.

## Key Learnings

* Different Claude models are optimized for different types of tasks.
* Sonnet is the most versatile model for daily student and developer workflows.
* Choosing the correct effort level improves efficiency and output quality.
* Providing detailed context leads to better AI-generated responses.
* Claude can be used as a learning assistant, coding mentor, research partner, and productivity coach.

## Outcome

Successfully generated a personalized Claude usage strategy, identified the best model and effort settings for my needs, and created an optimized workflow for learning, coding, research, and career development.
